    1. តើសំណុំ$A$ស្មើនឹងសំណុំ$B$ឬទេ?
    2. តើសំណុំ$A$ជាសំណុំរងនៃ$B$ឬទេ?
    3. តើសំណុំ$A$ជាសំណុំរងផ្ទាល់នៃ$B$ឬទេ?
  3. គេមានសំណុំ$A=\{4,3,6,7,1,9 \},$ $B=\{5,6,8,4 \}$ និង$C=\{5,8,6 \}$។

    ចូររក៖

    1. $A\cup B$
    2. $A\cap B$
    3. $A\cap C$
    4. $B\cap C$
    5. $B\cup C$
  4. គេមានសំណុំសកល$U=\{1,2,...,10 \},$ $A=\{4,3,6,7,1,9 \}$ និង$B=\{5,8,6,4 \}$។

    ចូររក៖

    1. $\overline{A}$
    2. $\overline{B}$
    3. $A\cup \overline{A}$
    4. $A\cap \overline{A}$
    5. $\overline{A}\cap B$
    6. $\overline{\overline{A}\cap B}$
  5. គេមានសំណុំសកល$U=\{x\mid x\,\text{ជាចំនួនគត់រ៉ឺឡាទីប:}\,1\lt x\lt 15 \},$និងសំណុំរងពីរនៃ$U$គឺ $A=\{x\mid x\,\text{ជាតួចែកវិជ្ជមាននៃ}\,12 \}$ និង$B=\{x\mid x\,\text{ជាតួចែកវិជ្ជមាននៃ}\,15 \}$ ។

    2. ចូររកសំណុំ$A\cap B$ និង$A\cup B$ ។
    3. ចូររកសំណុំរងបំពេញនៃសំណុំ$A$ ។
    4. ចូររកសំណុំ$\overline{A\cup B}$ ។
